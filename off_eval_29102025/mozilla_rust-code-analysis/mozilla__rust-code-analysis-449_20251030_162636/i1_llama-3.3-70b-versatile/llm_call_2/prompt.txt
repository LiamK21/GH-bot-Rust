Replace tree-sitter submodules with Rust bindings
Some of the `tree-sitter` grammars can be used as `Cargo.toml` dependencies. In this way, some submodules can be removed.

Pros
- A cleaner repository
- Avoid breaks during the update of a tree-sitter grammar due to some problems in handling submodules
- Less options and commands to clone the project

Cons 
- Impossibility to change the grammars directly in this repo, the grammars have to be modified upstream
