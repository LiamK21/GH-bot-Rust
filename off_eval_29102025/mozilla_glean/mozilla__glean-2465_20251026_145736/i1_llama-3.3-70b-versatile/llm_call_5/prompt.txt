TimingDistribution trait doesn't expose the accumulate_samples API
The [TimingDistribution trait](https://github.com/mozilla/glean/blob/main/glean-core/src/traits/timing_distribution.rs) doesn't expose the `accumulate_samples` or `accumulate_raw_samples` API functions and we are starting to see [use cases](https://phabricator.services.mozilla.com/D175589?id=707118#inline-974872) where we could use this.
