Issue:
<issue>
Glean should report windows build number along the os version
Right now Glean will report os version "10.0" for all versions of Windows 10 and upwards (if the surrounding application is build for that).

That's not enough to identify e.g. Windows 11. For that one also needs the build number. Glean should probably report that as part of the os version.
Glean uses `whatsys`, which already uses `GetVersionEx` to get the information, which includes the build number.
</issue>
